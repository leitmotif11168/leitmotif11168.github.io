Rules notes (tap to expand)
  • All listed cards use replacement effects. For a single event, each applies at most once.
  • Peregrin Took: add one Food to the event if any tokens would be created.
  • Academy Manufactor: if the event would create N total Food/Clue/Treasure, replace those with N of each (Food + Clue + Treasure).
  • Chatterfang: add Squirrels equal to the total number of tokens the event would create.
  • Parallel Lives / Doubling Season: each doubles all tokens from the event; together they quadruple them.
